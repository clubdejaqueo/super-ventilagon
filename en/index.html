Guide your spaceship,
avoiding a labyrinth made of light.

The second game by Club de Jaqueo,
inspired by Super Hexagon (2012).

An Arduino and 32 leds,
spinning at 1000RPM

All the game code is in an Arduino, bolted to the propeller of a big ass fan.

A pair of Wixels
link the propeller and the base

The base uses a Raspberry Pi to play the music and sound effects.

All the software and hardware
are free and open

You can build one by visiting our GitHub page.
